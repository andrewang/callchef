## 简单概述
这是一个线上召唤大厨的app

## 核心功能
1. 支持订餐；
2. 选择订餐后,厨师会第一时间联系您
3. 支持评分、分享、拨打客服电话
 
## 更改dbconfig.php中的数据库信息即可

## final.apk为安卓版的召唤大厨,安装到手机上即可查看

## chef.sql为这个app所需的sql,导入即可
